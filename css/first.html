1. Inline CSS

This is Inline CSS (written directly in the tag)
✅ Good for quick changes to one element only.
❌ Not reusable. ❌ Clutters HTML. ❌ Hard to maintain in large projects.

2. Internal CSS

This is Internal CSS (written in a <style> tag in the <head>)
✅ Good for styling an entire page.
✅ Centralized in one spot per file.
❌ Not reusable across multiple HTML files.

3. External CSS

This is External CSS (written in a separate .css file)
✅ Best for large websites.
✅ Keeps HTML clean.
✅ Reusable across multiple pages.
❌ Requires extra file and link setup.
